A lightweight, static Student Management System built with HTML, Bootstrap and vanilla JavaScript. Data is persisted in the browser using localStorage and the UI follows an elegant, minimalistic design with a top navigation and left sidebar.

Features
Dashboard overview with summary cards:
Total Students count
Total Courses count
Left sidebar + top navigation consistent across pages
Students page:
List students in a responsive table
Add student (separate form)
Delete student
Pre-seeded with 10 sample students on first load
Courses page:
List courses in a table
Add course (modal)
Delete course
Pre-seeded with 10 sample courses on first load
Counts update across pages (localStorage + storage-event notification)
Minimal, elegant styling with shared rules in style.css and page-level refinements
Accessible, responsive layout suited for desktop and smaller screens
